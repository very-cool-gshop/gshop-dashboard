# GShop Dashboard

電商後台管理系統，基於 [Nuxt UI](https://ui.nuxt.com) 建構，提供客戶管理、商品管理、訂單管理、排程任務等功能。

## 功能

- 數據總覽（營收、訂單、客戶統計）
- 客戶管理
- 商品管理與商品類別
- 訂單管理
- 排程任務
- 設定（一般、安全性）
- 側邊欄收合、深色模式

## Setup

```bash
npm install
```

## Development Server

```bash
npm run dev
```

## Production

```bash
npm run build
npm run preview
```
